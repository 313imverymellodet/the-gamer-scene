# Game Pass Starter Edition Leaked — 50+ Games and Cloud Streaming Bundled Into Discord Nitro

The leaks keep coming for Xbox's Game Pass strategy, and this one is significant. A post from **Discord Previews** on April 23 — corroborated by datamining from Xbox Cloud Gaming developer **redphx** — reveals a new **Game Pass Starter Edition** being tested as a perk for **Discord Nitro** subscribers. If confirmed, Discord Nitro users at the standard $9.99/month tier would receive access to a curated Game Pass library at no additional cost.

This is the "Xbox and Discord partnership" that CEO Asha Sharma had teased was coming. Now we have the details — or at least, what the leak shows.

## What the Starter Edition Includes

According to the Discord Previews post, Game Pass Starter Edition as a Nitro perk includes:

- **Access to 50+ games** from the Game Pass library (a curated subset, not the full catalog)
- **10 hours of cloud game streaming per month**
- **Xbox Rewards** access

The specific 50+ game selection hasn't been confirmed, but based on the structure of similar curated tiers at other subscription services, first-party titles like Grounded, Halo: The Master Chief Collection, Sea of Thieves, Stardew Valley, and Fallout 4 would be the most likely inclusions — solid, evergreen titles that demonstrate the value of the catalog without giving away the full library.

## The Pricing Context

Here's where it gets interesting. Discord Nitro currently costs **$9.99/month**. Xbox Game Pass Essential — the cheapest standalone tier — also costs **$9.99/month** and gives you access to just over 50 games plus unlimited cloud gaming.

On paper, that makes the standalone Essential tier the better pure gaming value: you get unlimited cloud streaming instead of the 10-hour cap, and you don't need a Discord subscription to access it. The argument for the Starter Edition is that it's **additive** — if you're already paying for Discord Nitro for its other features (server boosts, animated avatars, higher upload limits), you're now getting 50+ games and cloud gaming time on top at no additional cost.

The codename **"Duet"** that appeared in the Windows Central datamine suggests this bundle is being designed as a combined offering, not a replacement for existing tiers.

## The Bigger Picture: Xbox's Subscription Restructuring

This leak doesn't exist in isolation. Over the past month, Xbox under Asha Sharma has undergone a rapid-fire series of Game Pass changes:

**Price cut:** Game Pass Ultimate dropped from $29.99 to $22.99. PC Game Pass dropped from $16.99 to $13.99.

**Call of Duty removed from day one:** Future CoD releases will appear on Game Pass one year after launch rather than day one. A significant value reduction for the Ultimate tier that coincided with the price cut.

**"We Are Xbox" memo:** Yesterday's public commitment to making Game Pass more flexible and affordable.

**Starter Edition testing:** Today's leak.

The picture emerging is a company moving from a single premium subscription toward a tiered ecosystem — a model similar to how Spotify operates with free, individual, and family tiers, or how Apple has approached Apple One bundles. The Starter Edition fits into that model as a low-friction entry point that gets players into the Xbox ecosystem through an existing subscription they may already hold.

Whether a 10-hour monthly cloud cap is enough to convert casual Discord Nitro users into full Game Pass subscribers is the question Microsoft's data will eventually answer.

## Is This a Good Deal?

Purely on value, for existing Nitro subscribers: yes, obviously. You're getting 50+ games and cloud streaming time you wouldn't otherwise have at no additional cost. For players considering which subscription to get for gaming specifically: it depends on how much you value cloud gaming flexibility. The 10-hour cap is limiting — a single long gaming session could consume it. The unlimited cloud streaming of Game Pass Essential at the same price point is more practical for regular players.

The most interesting scenario is what this does for Discord's value proposition. Nitro has historically been a product for Discord power users who want premium server features. Bundling Game Pass access turns it into a meaningful gaming subscription in its own right — which could meaningfully expand Nitro's subscriber base among players who weren't previously considering it.

**Xbox Game Pass Starter Edition has not been officially announced. This information is based on a Discord Previews leak corroborated by datamining.**
